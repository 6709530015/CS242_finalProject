# CS242_finalProject
The final project of CS242 Python Programming

1. ที่มาของปัญหา (pain point / ปัญหา ที่เราต้องการแก้ไขคือ?)
นักศึกษาพลาดวันสำคัญ เช่น วันสอบ วันนำเสนอ หรือวันส่งงาน

2. แนวทางการแก้ปัญหาที่กลุ่มนักศึกษานำเสนอ (กลุ่มเราพัฒนาอะไร?)
ระบบรวบรวมนัดหมาย โดยมี Requirement ดังนี้
เพิ่มรายละเอียดนัดหมาย
ลบนัดหมายออกจากรายการ
แสดงผลวันนัดหมายทั้งหมดที่กำลังจะมาถึง
แสดงผลสถานะว่านัดหมายเลยกำหนดการไปแล้วหรือไม่
นับจำนวนวันก่อนถึงนัดหมาย
แสดงผลการแจ้งเตือนเมื่อใกล้ถึงวันนัดหมายแต่ละงาน
แสดงผลปฏิทินเพื่อดูภาพรวมจำนวนนัดหมายแต่ละเดือน (เรียก API Google Calendar)
วิเคราะห์จำนวนนัดหมายที่มีต่อเดือน

3. ประโยชน์ที่จะได้รับหากปัญหาได้รับการแก้ไขโดยใช้ solution ที่นักศึกษานำเสนอ 
ผู้ใช้สามารถมองเห็นภาพรวมวันนัดหมายที่มี
เพิ่มประสิทธิภาพในการวางแผนเวลาชีวิตมากขึ้น
ผู้ใช้ไม่พลาดนัดหมายสำคัญ

4. สถาปัตยกรรมระบบและแนวคิดการออกแบบ
Web UI - Flask framework
Server - Python OOP
Database - SQLite
Data / Library Integration - Request, Pandas or Matplotlib 
External Integration - Google Calendar, SQLite

5. Class Diagram (UML)


6. ผลการดำเนินงาน
